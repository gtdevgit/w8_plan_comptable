Plan comptable est une application produite par Pédalo production pour le store Windows 8. Elle présente le plan comptable général français. Elle permet de naviguer aisément dans l’arborescence des comptes comptables. Avec la fonction de recherche intégrée à l’application, il est possible de retrouver les comptes à partir de leur intitulé ou de leur racine. Cette application est gratuite et sans publicité.
